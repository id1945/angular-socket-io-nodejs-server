## Clone APP
```
git clone https://github.com/id1945/angular-socket-io-nodejs-server.git
```

## Server APP
```
http://localhost:3000
cd angular-socket-io-nodejs-server
npm install
npm start
```

## Angular APP
```
cd admin
npm install
npm start
```
